# Eye Health PDF Viewer 👁️📖

A sleek, web-based PDF viewer specifically designed to reduce eye strain during long study sessions. Instead of staring at glaring white backgrounds, this viewer automatically converts the background of any PDF into a soothing, customizable matte grey while keeping the text perfectly readable.

👉 **[Live Demo / Use the App Here!](https://anindya-jana.github.io/Eye-Health-PDF-Viewer/)**

## Features ✨

- **Eye-Care Rendering**: Uses advanced CSS `mix-blend-mode` to intelligently turn white backgrounds into a calm grey.
- **Customizable Page Tone**: A built-in color picker allows you to adjust the specific shade of the background (e.g., darker grey, warm sepia) to suit your environment and time of day.
- **Multi-Tab Support**: Open multiple PDFs at once and seamlessly switch between them using the tabbed interface.
- **Client-Side Processing**: Built with [PDF.js](https://mozilla.github.io/pdf.js/). Everything is processed locally in your browser for maximum privacy and speed. No files are ever uploaded to a server.

## How to Use 🚀

1. Open the [Live Webpage](https://anindya-jana.github.io/Eye-Health-PDF-Viewer/).
2. Click the **Open PDF** button.
3. Select one or more PDF files from your computer.
4. Enjoy reading without the eye strain! You can adjust the background color using the "Page Tone" color picker in the top left.

## Running Locally 💻

If you'd like to run this on your own machine:
1. Clone this repository.
2. Simply double-click the `index.html` file to open it in your web browser. No server setup is required!
